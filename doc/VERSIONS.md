# 版本切换说明

本项目支持新旧两个版本的灯效，可以通过编译宏切换。

## 切换版本

编辑 `main.c` 文件中的宏定义：

```c
#define USE_NEW_EFFECTS  1   // 1 = 新版本, 0 = 旧版本
```

## 新版本功能 (USE_NEW_EFFECTS = 1)

### 7种灯效模式

| 编号 | 模式 | 说明 | 按键操作 |
|------|------|------|----------|
| 0 | 慢闪模式 | 500ms切换亮灭 | 短按切换颜色（10色循环） |
| 1 | 快闪模式 | 100ms切换亮灭 | 短按切换颜色（10色循环） |
| 2 | 十色呼吸 | 自动呼吸，呼吸完自动换下一色 | 短按切换到下一模式 |
| 3 | 慢闪幻彩 | 每闪一次自动换色 | 短按切换到下一模式 |
| 4 | 快闪幻彩 | 快速闪，每闪一次自动换色 | 短按切换到下一模式 |
| 5 | 警示模式 | 红蓝快闪交替（50ms） | 短按切换到下一模式 |
| 6 | 跑马模式 | 每颗灯不同颜色跑马，转2圈后逐个熄灭 | 短按切换到下一模式 |

### 10种颜色

红、绿、蓝、黄、紫、青、橙、粉、黄绿、白

### 按键功能

- **短按** (< 1.5秒):
  - 慢闪/快闪模式：切换当前颜色
  - 其他模式：切换到下一个灯效模式
- **长按** (>= 1.5秒): 开/关机

## 旧版本功能 (USE_NEW_EFFECTS = 0)

保留原有的7种灯效：呼吸、闪烁、追逐、扩散、跑马、堆叠、警示

## 编译

切换版本后重新编译项目即可。

## 文件说明

- `effects.c/h` - 旧版本灯效
- `effects_new.c/h` - 新版本灯效
- `main.c` - 主程序（包含版本切换宏）
