# 版本切换指南

## 快速切换

### 使用新版本（当前配置）
项目文件 `8BIT-RGB.scw` 已配置为只编译新版本：
```
SourceFile=effects_new.c  (已启用)
SourceFile=effects.c       (已禁用)
```

`main.c` 中的设置：
```c
#define USE_NEW_EFFECTS  1
```

### 切换到旧版本

1. **修改项目文件** `8BIT-RGB.scw`：
   ```
   [FILE]
   SourceFile=effects.c       ; 启用旧版本
   ;SourceFile=effects_new.c  ; 禁用新版本（前面加分号注释）
   ```

2. **修改 main.c** 第4行：
   ```c
   #define USE_NEW_EFFECTS  0
   ```

3. **重新编译项目**

## 新版本灯效（7种）

| 编号 | 模式 | 说明 | 按键操作 |
|------|------|------|----------|
| 0 | 慢闪 | 500ms闪烁 | 短按换10色 |
| 1 | 快闪 | 100ms闪烁 | 短按换10色 |
| 2 | 十色呼吸 | 自动呼吸换色 | 短按切换模式 |
| 3 | 慢闪幻彩 | 500ms闪自动换色 | 短按切换模式 |
| 4 | 快闪幻彩 | 100ms闪自动换色 | 短按切换模式 |
| 5 | 警示 | 红蓝快闪50ms | 短按切换模式 |
| 6 | 跑马 | 单颗跑马→转2圈→逐个灭 | 短按切换模式 |

**10色**：红 绿 蓝 黄 紫 青 橙 粉 黄绿 白

---

## 文件说明

- **effects_new.c/h** - 新版本灯效（7种模式）
- **effects.c/h** - 旧版本灯效（保留，随时可用）
- **config.h** - 统一的配置和类型定义
- **main.c** - 主程序（包含版本切换宏）
