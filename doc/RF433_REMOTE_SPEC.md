# RF433 遥控器功能说明

## 遥控器按键功能

| 按键 | 功能 | 说明 |
|------|------|------|
| **OFF** | 关机 | 关闭所有LED，进入睡眠模式 |
| **ON** | 开机 | 唤醒并恢复上次状态，默认混白常亮 |
| **TIME** | 定时 | 循环切换: 4H → 6H → 8H → 无定时 |
| **DIM** | 亮度 | 循环切换亮度等级 (4档) |
| **TIME OFF** | 取消定时 | 关闭定时功能 |

## MODE 模式切换

| 模式 | 效果 | 颜色说明 |
|------|------|----------|
| **MODE 1** | 常亮模式 | 单色常亮，按键切换10种颜色 |
| **MODE 2** | 慢闪模式 | 单色慢闪，按键切换10种颜色 |
| **MODE 3** | 快闪模式 | 单色快闪，按键切换10种颜色 |
| **MODE 4** | 呼吸模式 | 十色自动呼吸渐变 |
| **MODE 5** | 慢闪幻彩 | 慢闪，每次换一种颜色 |
| **MODE 6** | 快闪幻彩 | 快闪，每次换一种颜色 |
| **MODE 7** | 警示模式 | 红/蓝快闪交替 |
| **MODE 8** | 跑马模式 | 单颗跑马，彩色循环 |

## 10种颜色 (MODE 1-3 使用)

| 编号 | 颜色 | RGB值 |
|------|------|-------|
| 0 | 红 | (255, 0, 0) |
| 1 | 绿 | (0, 255, 0) |
| 2 | 蓝 | (0, 0, 255) |
| 3 | 黄 | (255, 255, 0) |
| 4 | 紫 | (255, 0, 255) |
| 5 | 青 | (0, 255, 255) |
| 6 | 橙 | (255, 128, 0) |
| 7 | 粉 | (255, 0, 128) |
| 8 | 黄绿 | (128, 255, 0) |
| 9 | 混白 | (255, 255, 255) ← 默认 |

## 按键开关 (机身上)

| 操作 | 功能 |
|------|------|
| 长按 2 秒 | 开机/关机 |
| 短按 | 切换颜色 (MODE 1-3) 或 切换模式 |

## 震动检测

| 状态 | 动作 |
|------|------|
| 车行进中 | 灯自动亮起 |
| 车静止 | 30秒后灯自动熄灭 |

## 亮度等级

| 等级 | 亮度系数 | 说明 |
|------|---------|------|
| 1 | 100% | 高亮 |
| 2 | 75% | 中高 |
| 3 | 50% | 中等 |
| 4 | 25% | 低亮 |

## 出厂设置

- 模式: 常亮 (MODE 1)
- 颜色: 混白
- 亮度: 高亮
- 定时: 无
- 震动检测: 使能

## 亮度与电池关联

| 电池电压 | 最大亮度 | 说明 |
|---------|---------|------|
| 4.2V+ | 100% | 电池满电 |
| 4.0V-4.2V | 86% | 电量良好 |
| 3.7V-4.0V | 70% | 电量中等 |
| 3.5V-3.7V | 50% | 电量偏低 |
| <3.5V | 31% | 电量不足 |

| TIME 6H | 0x04 | MODE 8 | 0x17 |
| TIME 8H | 0x05 | 颜色+ | 0x20 |
| TIME OFF | 0x06 | 颜色- | 0x21 |
| DIM | 0x07 | RESET | 0xF0 |
| MODE 1 | 0x10 | | |
| MODE 2 | 0x11 | | |
| MODE 3 | 0x12 | | |
| MODE 4 | 0x13 | | |
