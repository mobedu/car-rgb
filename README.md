
# SC8F072 WS2812B LED 驱动项目

基于 SC8F072 单片机的 WS2812B 全彩 LED 驱动程序，支持多种亮灯效果，通过按键切换。

## 硬件连接

| 单片机引脚 | 功能          | 说明                  |
|-----------|---------------|-----------------------|
| RB1       | WS2812B 数据  | 接 LED 灯带 DIN 引脚  |
| RA0       | 按键输入      | 按键另一端接 GND      |
| VDD       | 电源正极      | 2.5V ~ 5.5V           |
| VSS       | 电源负极      | GND                   |

**注意**: WS2812B 电源电压建议 4.5V~5V，如果单片机供电 2.5V，可能需要电平转换电路。

## 功能特点

- 支持 4~15 颗 WS2812B LED (可配置)
- 7 种 LED 效果:
  1. 彩虹渐变 (EFFECT_RAINBOW)
  2. 呼吸灯 (EFFECT_BREATH)
  3. 流水灯 (EFFECT_FLOW)
  4. 静态红色 (EFFECT_STATIC_RED)
  5. 静态绿色 (EFFECT_STATIC_GREEN)
  6. 静态蓝色 (EFFECT_STATIC_BLUE)
  7. 闪烁 (EFFECT_BLINK)
- 按键消抖扫描
- 长按重置到第一个效果
- 低功耗设计 (使用 SLEEP 模式)

## 配置说明

在 `config.h` 中修改以下配置:

```c
#define LED_COUNT   8      // LED 灯珠数量 (4-15)
#define USE_16MHZ  1       // 1=使用16MHz, 0=使用4MHz
```

## 芯片配置字建议

```
WDT:    关闭 (或开启并确保所有路径喂狗)
LVR:    2.0V 或 2.5V
时钟:   内部 RC 振荡
代码保护: 关闭 (开发阶段)
```

## 编译说明

1. 打开 SC8F072 IDE
2. 新建工程
3. 手动添加以下文件到工程 (右键 -&gt; Add File):
   - main.c
   - ws2812.c
   - effects.c
   - key.c
   - config.h
   - ws2812.h
   - effects.h
   - key.h
4. 选择芯片型号 SC8F072
5. 配置时钟为内部 16MHz
6. 编译并烧录

## 时序说明

**WS2812B 协议要求** (16MHz 系统):
- 0码: 高 400ns, 低 850ns
- 1码: 高 800ns, 低 450ns
- 复位: 低电平 &gt; 50us

**指令周期**: 16MHz 系统使用 4T 模式时，单指令周期 = 250ns

## 注意事项

1. **中断关闭**: WS2812B 发送数据时会关闭中断，确保时序准确
2. **供电电压**: WS2812B 推荐 5V 供电，数据信号高电平应接近 VDD
3. **电平兼容**: 如果单片机 2.5V 供电而 LED 5V 供电，建议加电平转换
4. **WDT**: 如果开启看门狗，确保所有阻塞路径都喂狗
5. **ANSEL**: 所有用作数字 IO 的引脚必须清除对应的 ANSEL 位
